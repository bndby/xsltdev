# lang

| Тип    | Обязательный |
| ------ | ------------ |
| String | Нет          |

Параметр **`lang`** определяет основной язык для значений направленных свойств манифеста, и вместе с [`dir`](dir.md) определяет их направленность.

## Пример

```json
"lang": "en-US"
```
