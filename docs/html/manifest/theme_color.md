# theme_color

| Тип    | Обязательный |
| ------ | ------------ |
| String | Нет          |

Параметр **`theme_color`** определяет цвет темы по умолчанию для приложения.

Иногда влияет на то, как приложение отображается ОС (например, в переключателе задач Android цвет темы окружает приложение).

## Пример

```json
"theme_color": "red"
```
